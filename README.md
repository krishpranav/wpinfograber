# wpinfograber
A simpler perl tool for getting info about a wp website

# Steps To Run This Tool On Linux:
- bash install.sh
- git clone https://github.com/krishpranav/wpinfograber
- cd wpinfograber
- sudo chmod +x *
- perl wpgrab.ph

# Steps To Run This Tool On Android(Termux):
- bash install.sh
- git clone https://github.com/krishpranav/wpinfograber
- cd wpinfograber
- sudo chmod +x *
- perl wpgrab.ph
